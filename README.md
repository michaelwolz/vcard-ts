# vcard-ts

A modern TypeScript library with browser and Node.js support.

## Installation

```bash
npm install vcard-ts
```

## Usage

```typescript
import { greet } from 'vcard-ts';

console.log(greet('World'));
```

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Build the library
npm run build

# Lint code
npm run lint

# Format code
npm run format
```
